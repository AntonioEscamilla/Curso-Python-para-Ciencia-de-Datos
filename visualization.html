Cap√≠tulo 5: Visualizaci√≥n de Datos en Python

La Importancia de la Visualizaci√≥n

La visualizaci√≥n de datos es una herramienta fundamental en el proceso de an√°lisis de datos. Una buena visualizaci√≥n puede revelar patrones, tendencias y anomal√≠as que podr√≠an pasar desapercibidos en tablas de datos crudos. Como dijo Edward Tufte, uno de los pioneros en visualizaci√≥n de informaci√≥n:

"La excelencia gr√°fica es la que da al espectador la mayor cantidad de ideas en el menor tiempo con la menor cantidad de tinta en el menor espacio."

En la ciencia de datos, la visualizaci√≥n cumple varias funciones cr√≠ticas:

  • Exploraci√≥n de datos: Permite identificar r√°pidamente patrones, tendencias y valores at√≠picos durante el an√°lisis exploratorio.
  • Comunicaci√≥n de resultados: Facilita la transmisi√≥n eficaz de hallazgos a diferentes audiencias, desde t√©cnicas hasta no t√©cnicas.
  • Toma de decisiones: Proporciona informaci√≥n visual que ayuda a los stakeholders a tomar decisiones informadas.
  • Validaci√≥n de modelos: Permite evaluar visualmente el rendimiento y las predicciones de los modelos de machine learning.

Bibliotecas de Visualizaci√≥n en Python

Python ofrece un ecosistema rico de bibliotecas de visualizaci√≥n, cada una con sus propias fortalezas y caracter√≠sticas. En este cap√≠tulo, exploraremos las tres m√°s importantes:

Las bibliotecas que veremos se construyen sobre diferentes niveles de abstracci√≥n. Matplotlib ofrece control detallado, Seaborn simplifica la creaci√≥n de visualizaciones estad√≠sticas, y Plotly proporciona gr√°ficos interactivos para la web.

Matplotlib: La Biblioteca Fundamental

Matplotlib es una biblioteca de visualizaci√≥n completa para Python, inspirada en los gr√°ficos de MATLAB. Fue creada por John D. Hunter en 2003 y se ha convertido en la base sobre la cual se construyen muchas otras bibliotecas de visualizaci√≥n en Python como Seaborn, pandas plotting y plotly.

Matplotlib ofrece un control detallado sobre cada aspecto de un gr√°fico, desde los elementos m√°s b√°sicos hasta personalizaciones avanzadas, lo que la hace extremadamente flexible pero a veces verbosa para tareas complejas.

Instalaci√≥n de Matplotlib

Antes de comenzar a crear visualizaciones, necesitamos instalar Matplotlib. Lo podemos hacer f√°cilmente con pip:

# Instalar Matplotlib con pip
pip install matplotlib

Si est√°s utilizando Anaconda, Matplotlib ya viene incluido en la distribuci√≥n. Para verificar la versi√≥n instalada, puedes ejecutar:

# Verificar la versi√≥n de Matplotlib
import matplotlib
print(matplotlib.__version__)
3.7.1

Importar la Biblioteca

Existen diferentes formas de importar Matplotlib. La m√°s com√∫n es importar el subm√≥dulo pyplot, que proporciona una interfaz similar a MATLAB:

# Forma est√°ndar de importar Matplotlib
import matplotlib.pyplot as plt

# Para mostrar gr√°ficos en notebooks de Jupyter
%matplotlib inline

# Para gr√°ficos interactivos en notebooks
%matplotlib widget  # Requiere ipympl
El alias plt es una convenci√≥n ampliamente utilizada al importar matplotlib.pyplot. Facilita la escritura de c√≥digo y lo hace m√°s consistente con las pr√°cticas de la comunidad.

Comandos B√°sicos de Matplotlib

Veamos c√≥mo crear un gr√°fico simple con Matplotlib:

# Importar las bibliotecas necesarias
import matplotlib.pyplot as plt
import numpy as np

# Crear datos de ejemplo
x = np.linspace(0, 10, 100)  # 100 puntos entre 0 y 10
y = np.sin(x)                # Funci√≥n seno

# Crear un gr√°fico simple
plt.figure(figsize=(8, 4))   # Tama√±o de la figura en pulgadas
plt.plot(x, y)               # Crear la l√≠nea
plt.title('Funci√≥n Seno')    # A√±adir t√≠tulo
plt.xlabel('x')              # Etiqueta del eje x
plt.ylabel('sen(x)')         # Etiqueta del eje y
plt.grid(True)               # Mostrar cuadr√≠cula
plt.show()                   # Mostrar el gr√°fico
Gr√°fico simple de la funci√≥n seno creado con Matplotlib

Matplotlib tiene dos interfaces principales:

  • Interfaz pyplot (procedural): Similar a MATLAB, donde las funciones como plt.plot() act√∫an sobre la figura "actual".
  • Interfaz orientada a objetos: M√°s flexible y expl√≠cita, donde se trabaja directamente con objetos como Figure y Axes.

El ejemplo anterior utiliza la interfaz pyplot. Veamos el mismo gr√°fico usando la interfaz orientada a objetos:

# Usando la interfaz orientada a objetos
fig, ax = plt.subplots(figsize=(8, 4))
ax.plot(x, y)
ax.set_title('Funci√≥n Seno')
ax.set_xlabel('x')
ax.set_ylabel('sen(x)')
ax.grid(True)
plt.show()
Para proyectos complejos, se recomienda utilizar la interfaz orientada a objetos, ya que proporciona m√°s control y claridad, especialmente cuando se trabaja con m√∫ltiples subplots o gr√°ficos personalizados.

Personalizaci√≥n de Gr√°ficos

Matplotlib ofrece un control detallado sobre pr√°cticamente todos los elementos de un gr√°fico. Veamos c√≥mo personalizar diferentes aspectos:

T√≠tulo

El t√≠tulo es uno de los elementos m√°s importantes de un gr√°fico. Puedes personalizarlo con diferentes opciones:

# Personalizaci√≥n del t√≠tulo
plt.figure(figsize=(8, 4))
plt.plot(x, y)

# T√≠tulo b√°sico
plt.title('Funci√≥n Seno')

# T√≠tulo con m√°s opciones
plt.title('Funci√≥n Seno: f(x) = sen(x)', 
          fontsize=16,              # Tama√±o de fuente
          fontweight='bold',        # Peso de la fuente: 'normal', 'bold', etc.
          color='navy',             # Color del texto
          loc='center',             # Posici√≥n: 'center', 'left', 'right'
          pad=20)                   # Espacio en puntos entre el t√≠tulo y el gr√°fico

plt.plot(x, y)
plt.grid(True)
plt.show()
Gr√°fico con t√≠tulo personalizado

Nombres de los Ejes

Etiquetar correctamente los ejes es fundamental para que un gr√°fico sea comprensible:

# Personalizaci√≥n de etiquetas de ejes
plt.figure(figsize=(8, 4))
plt.plot(x, y)

# Etiquetas b√°sicas
plt.xlabel('Tiempo (s)')
plt.ylabel('Amplitud')

# Etiquetas con m√°s opciones
plt.xlabel('Tiempo (s)', 
          fontsize=12,
          color='darkred',
          labelpad=10)  # Espacio entre la etiqueta y el eje

plt.ylabel('Amplitud', 
          fontsize=12,
          color='darkred',
          labelpad=10)

plt.title('Funci√≥n Seno')
plt.grid(True)
plt.show()
Gr√°fico con etiquetas de ejes personalizadas

Para a√±adir f√≥rmulas matem√°ticas en las etiquetas, puedes usar la sintaxis de LaTeX:

# Etiquetas con f√≥rmulas matem√°ticas (LaTeX)
plt.figure(figsize=(8, 4))
plt.plot(x, y)

plt.xlabel('√Ångulo ($\\theta$)')
plt.ylabel('$\\sin(\\theta)$')  # Notaci√≥n matem√°tica
plt.title('$f(\\theta) = \\sin(\\theta)$')

plt.grid(True)
plt.show()
Para usar LaTeX en etiquetas, encierra la f√≥rmula entre $. En cadenas de Python, necesitas escapar el backslash, por lo que \alpha se escribe como \\alpha.

Legend (Leyenda)

Las leyendas son esenciales cuando tienes m√∫ltiples series de datos en un gr√°fico:

# A√±adir leyendas a gr√°ficos
plt.figure(figsize=(8, 4))

# Graficar m√∫ltiples l√≠neas con etiquetas

plt.plot(x, np.cos(x), label='cos(x)')
plt.plot(x, np.exp(-x/3), label='$e^{-x/3}$')

# A√±adir leyenda b√°sica
plt.legend()

# O leyenda con m√°s opciones
plt.legend(
    loc='upper right',        # Ubicaci√≥n: 'best', 'upper right', 'center left', etc.
    frameon=True,             # Mostrar borde
    fancybox=True,            # Esquinas redondeadas
    shadow=True,              # Sombra
    fontsize=10,              # Tama√±o de fuente
    ncol=1                    # N√∫mero de columnas
)

plt.title('M√∫ltiples Funciones')
plt.xlabel('x')
plt.ylabel('y')
plt.grid(True)
plt.show()
Gr√°fico con m√∫ltiples l√≠neas y leyenda personalizada
Si no proporcionas el par√°metro label cuando creas un elemento en el gr√°fico, ese elemento no aparecer√° en la leyenda aunque llames a plt.legend().

Cuadr√≠cula (Grid)

La cuadr√≠cula puede ayudar a leer valores con mayor precisi√≥n y mejorar la legibilidad del gr√°fico:

# Personalizaci√≥n de la cuadr√≠cula
plt.figure(figsize=(8, 4))

M√°s Opciones de Personalizaci√≥n

Tama√±o de la Figura y DPI

El tama√±o y la resoluci√≥n de la figura son aspectos importantes, especialmente cuando necesitas gr√°ficos para publicaciones o presentaciones:

# Configuraci√≥n del tama√±o y resoluci√≥n
# figsize: (ancho, alto) en pulgadas
# dpi: puntos por pulgada (resoluci√≥n)

plt.figure(figsize=(8, 4), dpi=100)  # Figura de 8x4 pulgadas a 100 dpi
plt.plot(x, np.sin(x))
plt.title('Figura de 8x4 pulgadas a 100 dpi')
plt.show()

plt.figure(figsize=(4, 3), dpi=150)  # Figura m√°s peque√±a con mayor resoluci√≥n
plt.plot(x, np.sin(x))
plt.title('Figura de 4x3 pulgadas a 150 dpi')
plt.show()
El tama√±o de la figura (figsize) afecta las proporciones y el espacio disponible para el gr√°fico, mientras que el DPI (dpi) controla la resoluci√≥n del gr√°fico exportado. Para proyectos web, 100 dpi suele ser suficiente, mientras que para publicaciones acad√©micas se recomienda 300 dpi o m√°s.

Par√°metros de las L√≠neas: Colores, Ancho y Tipos

Matplotlib ofrece muchas opciones para personalizar el aspecto de las l√≠neas en tus gr√°ficos:

# Personalizaci√≥n de l√≠neas
plt.figure(figsize=(10, 6))

# Diferentes estilos de l√≠nea

         color='blue',         # Color por nombre
         linestyle='-',        # Estilo de l√≠nea: s√≥lida

         marker='o',           # Marcador: c√≠rculo
         markersize=6,         # Tama√±o del marcador
         markerfacecolor='white',  # Color interior del marcador
         markeredgecolor='blue',   # Color del borde del marcador
         markeredgewidth=1.5,      # Ancho del borde del marcador
         alpha=0.7,            # Transparencia
         label='sin(x) - detallado')

# Forma simplificada usando la sintaxis abreviada
plt.plot(x, np.cos(x), 'r--', linewidth=2, label='cos(x) - simplificado')

# Tercera l√≠nea con otro estilo
plt.plot(x, np.sin(x) * np.cos(x), 'g-.', linewidth=1.5, label='sin(x)cos(x)')

plt.title('Diferentes Estilos de L√≠nea')
plt.xlabel('x')
plt.ylabel('y')
plt.legend()
plt.grid(True)
plt.show()
Gr√°fico con diferentes estilos de l√≠nea y marcadores

La sintaxis abreviada para definir l√≠neas combina color, marcador y estilo en un solo string:

# Demostraci√≥n de la sintaxis abreviada

    'b-', 'r--', 'g-.', 'c:', 'm-o', 'y-^', 'k-s', 
    'b--o', 'r-+', 'g:*', 'c-d', 'm--p', 'y-.h'
]

# Crear m√∫ltiples l√≠neas con diferentes estilos

    plt.plot(x, np.sin(x + i*0.5), estilo, label=f'Estilo: {estilo}')

plt.title('Demostraci√≥n de Estilos de L√≠nea')
plt.xlabel('x')
plt.ylabel('y')
plt.legend(ncol=2)  # Leyenda con 2 columnas
plt.grid(True)
plt.show()

Opciones comunes para la sintaxis abreviada

Colores 'b' (blue), 'g' (green), 'r' (red), 'c' (cyan), 'm' (magenta), 'y' (yellow), 'k' (black), 'w' (white)
Marcadores '.' (punto), 'o' (c√≠rculo), 'v' (tri√°ngulo abajo), '^' (tri√°ngulo arriba), 's' (cuadrado), '+' (m√°s), 'x' (equis), '*' (estrella)
L√≠neas '-' (s√≥lida), '--' (discontinua), '-.' (punto y raya), ':' (punteada)
Adem√°s de los nombres de colores predefinidos, puedes usar c√≥digos hexadecimales como '#FF5733', tuplas RGB como (0.2, 0.4, 0.6) o nombres de colores HTML como 'tomato' o 'cornflowerblue'.

Subplots: M√∫ltiples Gr√°ficos

Los subplots permiten crear m√∫ltiples gr√°ficos en una sola figura, lo que es √∫til para comparar diferentes conjuntos de datos o visualizar diferentes aspectos de los mismos datos.

Creaci√≥n B√°sica de Subplots

# Crear una cuadr√≠cula simple de subplots (2 filas, 2 columnas)
fig, axs = plt.subplots(2, 2, figsize=(10, 8))

# Graficar en cada subplot
axs[0, 0].plot(x, np.sin(x))
axs[0, 0].set_title('Seno')
axs[0, 0].grid(True)

axs[0, 1].plot(x, np.cos(x), 'g-')
axs[0, 1].set_title('Coseno')
axs[0, 1].grid(True)

axs[1, 0].plot(x, np.tan(x), 'r-')
axs[1, 0].set_title('Tangente')
axs[1, 0].set_ylim(-5, 5)  # Limitar el rango del eje y
axs[1, 0].grid(True)

axs[1, 1].plot(x, np.exp(x), 'm-')
axs[1, 1].set_title('Exponencial')
axs[1, 1].grid(True)

# Ajustar el espaciado entre subplots
plt.tight_layout()

# A√±adir un t√≠tulo para toda la figura
fig.suptitle('Funciones Matem√°ticas B√°sicas', fontsize=16, y=1.02)

plt.show()
Cuadr√≠cula de subplots 2x2 con diferentes funciones

Anotaciones de Texto

Las anotaciones de texto permiten destacar puntos espec√≠ficos o caracter√≠sticas importantes en un gr√°fico:

Diagramas de Caja (Boxplot)

Los boxplots muestran: el primer cuartil (Q1, 25%), la mediana (Q2, 50%), el tercer cuartil (Q3, 75%), los bigotes (t√≠picamente 1.5*IQR desde Q1 y Q3), y los valores at√≠picos como puntos individuales. Son excelentes para comparar distribuciones y detectar asimetr√≠as y valores extremos.
