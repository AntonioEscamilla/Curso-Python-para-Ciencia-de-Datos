Cap√≠tulo 2: NumPy - La Base del An√°lisis Num√©rico en Python

¬øQu√© es NumPy?

NumPy (Numerical Python) es la biblioteca fundamental para la computaci√≥n cient√≠fica en Python. Proporciona soporte para arrays y matrices multidimensionales, junto con una colecci√≥n de funciones matem√°ticas de alto nivel para operar con estos arrays de manera eficiente.

¬øPor qu√© NumPy es esencial para la ciencia de datos?

  • Eficiencia computacional: NumPy implementa algoritmos optimizados en C que ejecutan operaciones mucho m√°s r√°pido que el c√≥digo Python puro, especialmente para grandes conjuntos de datos.
  • Manipulaci√≥n de arrays: Ofrece estructuras de datos potentes y funciones para manipular datos num√©ricos, facilitando operaciones que ser√≠an complejas con las estructuras de datos nativas de Python.
  • Algebra lineal: Proporciona funciones para operaciones matriciales, factorizaciones, descomposiciones y m√°s, esenciales para muchos algoritmos de machine learning.
  • Integraci√≥n del ecosistema: NumPy es la base sobre la cual se construyen muchas otras bibliotecas para ciencia de datos como Pandas, SciPy, Matplotlib y scikit-learn.
  • Operaciones vectorizadas: Permite realizar operaciones en arrays completos sin necesidad de bucles expl√≠citos, lo que hace el c√≥digo m√°s limpio y mucho m√°s r√°pido.

En este cap√≠tulo, exploraremos los fundamentos de NumPy y aprenderemos a utilizar sus capacidades para manipular datos num√©ricos de manera eficiente.

Arrays en NumPy

Los arrays son la estructura de datos principal en NumPy. Podemos pensar en ellos como colecciones de elementos del mismo tipo organizados en una estructura de filas y columnas.

Slicing en Arrays

El slicing en NumPy sigue la sintaxis inicio:fin:paso y funciona en todas las dimensiones:

# Slicing en array unidimensional

# Sintaxis: array[inicio:fin:paso]

print(f"Elementos hasta el 4: {arr[:5]}")
print(f"Elementos desde el 6: {arr[6:]}")
print(f"Elementos con paso 2: {arr[1:8:2]}")
print(f"Array invertido: {arr[::-1]}")

# Slicing en array bidimensional
matriz = np.array([[1, 2, 3, 4],

Funciones Universales (ufuncs) en NumPy

NumPy tiene integradas muchas funciones universales (ufuncs), que son esencialmente operaciones matem√°ticas que se pueden aplicar a todo el array de manera vectorizada. Estas funciones operan elemento por elemento y son extremadamente eficientes.

Las categor√≠as principales de funciones universales incluyen:

  • Matem√°ticas: sqrt, exp, log, power, etc.
  • Trigonometr√≠a: sin, cos, tan, etc.
  • Funciones de comparaci√≥n: equal, greater_than, less_than, etc.
  • Funciones de punto flotante: isnan, isinf, etc.
